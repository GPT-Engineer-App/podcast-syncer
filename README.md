# podcast-syncer

A live podcasting Communities management system, users can integrating those popular social media platform like twitter space, Discord, Linkedin, Facebook, Youtube for sync the audio live streaming podcast show. then user only need to use this platform to connect all API, then sync the audio show to all account.

- React

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/podcast-syncer.git
cd podcast-syncer
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
